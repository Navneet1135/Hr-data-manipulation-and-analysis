# HR Data Manipulation and Analysis Using SQL

## Overview

This project focuses on HR data manipulation, validation, and analysis using SQL. The analysis is performed on employee master data and performance review data from ABC Inc. to generate meaningful workforce insights.

The project demonstrates how SQL can be used to clean, validate, transform, and analyze HR datasets to support data-driven decision-making in Human Resources.

---

## Datasets

### Employees Master Dataset

Contains employee information including:

- Employee ID
- Full Name
- Gender
- Department
- Job Title
- Employment Type
- Location
- Hire Date
- Monthly Salary (BDT)
- Manager ID

### Performance Reviews 2024 Dataset

Contains employee performance information including:

- Employee ID
- Review Period Start
- Review Period End
- Performance Rating
- Manager Score
- Peer Score
- Bonus Percentage
- Trainings Completed
- Leaves Taken
- Attrition Risk

---

## Project Objectives

- Perform data quality validation and cleaning.
- Analyze workforce distribution across departments.
- Evaluate salary trends and compensation fairness.
- Assess employee performance ratings.
- Identify attrition risk patterns.
- Analyze bonus allocation strategies.
- Examine training and leave behavior.
- Evaluate manager span of control.
- Analyze employee tenure and retention trends.

## Analysis Performed

- Median salary by department

### Performance Analysis
- Average performance rating by department
- Manager score versus peer score comparison

### Attrition Risk Analysis
- Attrition risk distribution

### Bonus Analysis
- Top employees by bonus amount
- Department-wise average bonus analysis

### Training and Leave Analysis
- Training completion trends
- Leave utilization patterns

### Manager Analysis
- Direct reports per manager
- Managers with large teams

### Tenure Analysis
- Employee tenure calculation
- Average tenure by department
